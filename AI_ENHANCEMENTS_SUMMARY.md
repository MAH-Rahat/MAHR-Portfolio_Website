## 🎨 Customization Options

### Adjust Background Intensity:
```jsx
// In App.jsx

// Subtle (Current):
<MatrixRain density={0.3} speed={1} opacity={0.1} />
<NeuralNetwork nodeCount={25} connectionOpacity={0.15} />

// Medium:
<MatrixRain density={0.5} speed={1.5} opacity={0.15} />
<NeuralNetwork nodeCount={35} connectionOpacity={0.25} />

// Strong:
<MatrixRain density={0.7} speed={2} opacity={0.2} />
<NeuralNetwork nodeCount={50} connectionOpacity={0.35} />
```

### Disable Features:
```jsx
// Comment out any feature you don't want:
{/* <MatrixRain ... /> */}
{/* <NeuralNetwork ... /> */}
{/* <AITerminal /> */}
```
