## 🤖 AI & Visual Effects

### Background Effects (Layered)
1. **Circuit Board Pattern** (`CircuitBoard.jsx`)
   - Procedurally generated circuit lines
   - Right-angle connections
   - Random circuit elements
   - Adjustable opacity and color

2. **Matrix Rain** (`MatrixRain.jsx`)
   - Falling Japanese characters
   - Adjustable density and speed
   - Authentic Matrix-style effect

3. **Neural Network** (`NeuralNetwork.jsx`)
   - Animated nodes with connections
   - Simulates neural network visualization
   - Dynamic connection lines

4. **Holographic Overlay** (`HolographicOverlay.jsx`)
   - Shifting grid pattern
   - Holographic/futuristic feel
   - Three intensity levels

5. **Scan Line** (`ScanLine.jsx`)
   - Moving horizontal line
   - Retro CRT monitor effect
   - Adjustable speed and opacity

6. **Data Streams** (`DataStream.jsx`)
   - Flowing data on both sides
   - Hex codes, operators, tech terms
   - Bidirectional flow

7. **Cursor Trail** (`CursorTrail.jsx`)
   - Colorful particles follow mouse
   - Natural fade and dispersion
   - Canvas-based animation

### Interactive Effects
1. **Glitch Text** (`GlitchText.jsx`)
   - RGB split effect
   - Periodic glitching
   - Used on hero name
   - Adjustable intensity

2. **Typing Animation** (`TypingAnimation.jsx`)
   - Multiple role titles
   - Smooth typing effect
   - Cursor blink animation

3. **Particle Background** (`ParticleBackground.jsx`)
   - Floating particles
   - Interactive on hover
   - Connects nearby particles

### Widgets
1. **AI Terminal** (`AITerminal.jsx`)
   - System initialization messages
   - Uptime counter
   - Status indicators
   - Bottom-right corner

2. **AI Stats Widget** (`AIStatsWidget.jsx`)
   - Neural Networks Trained
