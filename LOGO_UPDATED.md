## Technical Details

### Icon
- **Component**: `FaBrain` from react-icons
- **Size**: 28px
- **Color**: Primary blue with dark mode support

### Animations
- **Glow**: Drop-shadow animation (2s loop)
- **Pulse**: Scale and opacity animation (2s loop)
- **Hover**: Scale 1.1x on hover

### Text Styling
- **Name**: Gradient from primary to secondary color
- **Subtitle**: Small gray text, uppercase, letter-spaced
- **Font**: Bold for name, normal for subtitle

## Customization Options

### Change Icon
Replace `FaBrain` with other AI-related icons:
- `FaRobot` - Robot icon
- `FaMicrochip` - Chip/processor icon
- `FaNetworkWired` - Network icon
- `FaProjectDiagram` - Neural network diagram

### Change Subtitle
Edit line with "AI ENGINEER" to:
- "ML SPECIALIST"
- "AI DEVELOPER"
- "FULL-STACK AI"
- Or remove it entirely

### Adjust Colors
Modify the gradient colors:
```jsx
from-primary-600 to-secondary-600
```

Change to any Tailwind color combination!
