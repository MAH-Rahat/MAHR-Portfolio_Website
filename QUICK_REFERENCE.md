# 🚀 Quick Reference Card

## 🎮 Interactive Features

### Floating Action Button (FAB)
- **Location:** Bottom-left corner (rocket icon 🚀)
- **Actions:**
  - 🎮 Play Snake Game
  - 🏆 View Achievements
  - 💡 Easter Egg Hints

### Snake Game Controls
- **Arrow Keys:** Move snake
- **Space:** Pause/Resume
- **R:** Restart
- **Escape:** Close game

### Easter Egg Codes
Type these anywhere on the site:
- `↑↑↓↓←→←→BA` - Konami Code
- `hello` - Greeting
- `ai` - AI message
- `matrix` - Matrix reference
- `hack` - Hacker message
- `code` - Code poetry
- `dev` - Developer message
- `neural` - Neural joke

## 🔧 Customization

Edit `client/src/App.jsx` to adjust:

```jsx

<CircuitBoard opacity={0.05} />
<MatrixRain density={0.2} />

// Disable effects
{/* <CursorTrail /> */}
```
