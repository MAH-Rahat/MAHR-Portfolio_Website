# 🎮 How to Play the Snake Game

## Where is the Game?

The game is accessed through the **Floating Action Button (FAB)** on your website!

### Step-by-Step Guide:

1. **Look at the bottom-left corner** of your website
2. You'll see a **blue/purple gradient button with a rocket icon (🚀)**
3. **Click on the rocket button**
4. A menu will slide up with 3 options:
   - 🎮 **Play Snake** ← Click this one!
   - 🏆 Achievements
   - 💡 Easter Egg

5. Click "Play Snake" and the game will open in a modal!

## Game Controls

Once the game opens:

- **Arrow Keys (↑ ↓ ← →):** Move the snake
- **Space Bar:** Pause/Resume
- **R Key:** Restart game
- **Escape or X button:** Close game

## Troubleshooting

**Can't find the button?**
- Scroll to any part of the page
- Look at the **bottom-left corner**
- It's a circular button with a rocket icon
- It has a pulsing ring animation

**Button not working?**
- Make sure JavaScript is enabled
- Try refreshing the page
- Check browser console for errors

**Game not loading?**
- Click the rocket button
- Wait for the menu to appear
- Click "Play Snake"
- The game modal should open
